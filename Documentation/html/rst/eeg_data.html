eeg_data.DiscreteData module¶

class eeg_data.DiscreteData.DiscreteData(data, event_type_id, start=-1000, end=2000)[source]¶

Bases: object

Holds discrete data.

Each object of this class holds epochs extracted basing on exactly one event. Epochs are kept in a np.array (:var epochs) of respective Epoch objects, position in the array responds to the event id.

average_epochs(channels)[source]¶

Returns an arithmetic average value of brain activation over all epochs.

get_event_from_event_channel(channel_name, threshold=1000)[source]¶

Extracts information about an event from a given channel (:argument channel_name).

Extracts events in the simplest way – it treats as an event marker a value of signal that is higher than a given value (:keyword threshold). This function works best when in the data there is a channel that has only non-zero values when the event occurs (that is, a dedicated event channel).

Module contents¶

Contains data representation (in continuous and discrete form) as well as tools for importing data.

Reader – imports the EEG data from a file.
Returns them in a format that requires further processing by the ImportedData class to be readable for pytnam.
