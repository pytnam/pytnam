context.AppContext module¶

This is a class to be used by the application control logic. Every time a new analysis pipeline is created (i.e., usually, once per session)), AppContext is adjusted to contain: a PipelineInfo object with info about the currently performed sequence of analyses, a list of lxml Element objects representing subsequent steps of the analysis (pipeline nodes). Pipeline nodes/analysis steps can be added by using the add_pipeline_node method.
